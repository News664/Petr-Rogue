# Backgrounds

Scene backdrops rendered behind gameplay.

## Files

| File         | Size (px) | Aspect   | Transparency | Content                                   |
|--------------|-----------|----------|--------------|-------------------------------------------|
| `combat.png` | 1672×941  | ~16:9    | opaque       | Backdrop for the combat scene. Sprites and portraits are composited on top, so keep the composition uncluttered in the center where figures stand. |

## Notes

More backgrounds may be added later (e.g. per-act or per-scene variants). Follow
the same landscape, opaque convention; match the target region's aspect ratio
and provide the spec here when new files are introduced.
